## Microsoft offers sneak peek at new Windows features coming this fall
 - [https://www.pcworld.com/article/2004412/microsoft-offers-sneak-peek-at-new-windows-11-features-due-this-fall.html](https://www.pcworld.com/article/2004412/microsoft-offers-sneak-peek-at-new-windows-11-features-due-this-fall.html)
 - RSS feed: https://www.pcworld.com/index.rss
 - date published: 2023-07-21T18:56:43+00:00

<div id="link_wrapped_content">
<section class="wp-block-bigbite-multi-title"><div class="container"></div></section>



<p>Microsoft is moving several significant features into its latest Windows 11 Insider Beta Channel release today, a sign that it&rsquo;s committed to updates like RGB lighting controls, an updated File Explorer, and a subtle but nifty improvement in inking.</p>



<p>Microsoft released Windows 11 Insider Preview Build 22621.2050 and Build 22631.2050 to the Windows Insider Beta Channel today. Shipping them to users in the Beta Channel has historically been the first official signal that Microsoft has signed off on the features to eventually ship them. (The &ldquo;631&rdquo; build includes all of the features for testing, while the other holds some back.)</p>



<p>The &ldquo;new&rdquo; features may seem familiar, especially if you&rsquo;ve been following releases in the experimental Canary or Dev Channels, where Microsoft doesn&rsquo;t always bring new features to market if they test poorly. Microsoft hasn&rsquo;t said when these new features would be available, but with mentions of &ldquo;23H2&rdquo; already bubbling up, it seems likely that they&rsquo;ll appear in September or October.</p>



<p>Here&rsquo;s a look at what Microsoft is promising:</p>



<h2 class="wp-block-heading" id="a-new-modernized-file-explorer">A (new) modernized File Explorer</h2>



<p>Microsoft has revamped the File Explorer app previously&mdash;<a href="https://www.pcworld.com/article/1071395/heres-what-microsoft-plans-for-windows-11-22h2s-first-update.html">adding tabs</a>, a <a href="https://www.pcworld.com/article/1530047/a-pizza-icon-debuts-today-in-windows-file-explorer.html">pizza icon</a>, better visibility into available OneDrive storage, and deeper insights into file previews in the preview page. It&rsquo;s heading further down this path, in part because the new WinUI framework allows additional customization, Microsoft says.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-large"><img alt="Windows 11 23H2 File Explorer" class="wp-image-2004449" height="716" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/File-Explorer-1.jpg?quality=50&amp;strip=all&amp;w=1200" width="1200" /><figcaption class="wp-element-caption">An attractive carousel view will greet you on the File Explorer home view.</figcaption></figure><p class="imageCredit">Microsoft</p></div>



<p>File Explorer will show Recommended files in a carousel view from the Home view. You&rsquo;ll also see your OneDrive status integrated into File Explorer, and any cloud folders will be better integrated into Windows. Finally, the Details pane in File Explorer (<strong>Alt+Shift+P</strong>) will even show you related content and show recent activity on a file if it&rsquo;s been shared and/or modified.</p>



<p>Specifically, when a user selects a file in File Explorer and enables the pane, a modern pane will display contextual information about the file including file thumbnail, share status and button, file activity, related files and e-mails, and other information, Microsoft <a href="https://go.redirectingat.com/?id=111346X1569483&amp;url=https://blogs.windows.com/windows-insider/2023/07/21/announcing-windows-11-insider-preview-build-22621-2050-and-22631-2050/&amp;xcust=2-1-2004412-1-0-0&amp;sref=https://www.pcworld.com/feed" rel="nofollow">said</a>.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-large"><img alt="Windows 11 23H2 File Explorer" class="wp-image-2004446" height="1200" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/File-Explorer-3.jpg?quality=50&amp;strip=all&amp;w=600" width="600" /><figcaption class="wp-element-caption">File Explorer&rsquo;s new preview pane.</figcaption></figure><p class="imageCredit">Microsoft</p></div>



<h2 class="wp-block-heading" id="passkey-management">Passkey management</h2>



<p>Last month, <a href="https://www.pcworld.com/article/1969610/windows-hello-gets-new-passkey-powers.html">Microsoft began building in passkey support</a> within its Windows Insider Dev Channel. Passkeys replaces passwords on sites like BestBuy.com with the biometrics on your PC or phone&mdash;if a Web site like Amazon adds passkey support for the Web, you won&rsquo;t need to enter (or remember) your password. Instead, you&rsquo;ll use Windows Hello. </p>



<p>Microsoft is adding a Settings page (<em>Settings &gt; Accounts &gt; Passkeys</em>) where you can manage all of the passkeys for sites you&rsquo;ve visited. </p>



<p>This is a fantastic step forward for ease of use, but it will need industry support to move further ahead.</p>



<h2 class="wp-block-heading" id="unsafe-password-warnings">Unsafe password warnings</h2>



<p>If you <em>do</em> use passwords, though, as we all do, Microsoft is also warning you not to copy your password using cut and paste. Microsoft is adding a feature to Windows Security under App &amp; browser control &gt; Reputation-based protection &gt; Phishing protection to warn you on the unsafe habit of copying a password via copy and paste.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-full"><img alt="Windows 11 23H2 password reuse" class="wp-image-2004444" height="298" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Unsafe-password.jpg?quality=50&amp;strip=all" width="604" /></figure><p class="imageCredit">Microsoft</p></div>



<h2 class="wp-block-heading" id="dynamic-lighting-gets-a-stamp-of-approval">Dynamic Lighting gets a stamp of approval</h2>



<p>Given that Microsoft has already chosen to work with several PC accessories companies that include RGB, it seemed inevitable that the <a href="https://www.pcworld.com/article/1928248/dynamic-lighting-will-be-windows-tool-to-manage-your-pcs-bling.html">overarching controls for PC RGB lighting, known as Dynamic Lighting</a>, would appear in Windows. Normally, vendors often provide their own RGB lighting control software. Third-party apps also try and provide umbrella access. </p>



<p>Microsoft decided to bring it all within Windows, so you can make your PC as sparkly as possible. A new Dynamic Lighting settings page will provide a one-stop shop for all of these controls.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-full"><img alt="Windows 11 23H2 dynamic lighting" class="wp-image-2004445" height="729" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Dynamic-Lighting-1.jpg?quality=50&amp;strip=all" width="1024" /><figcaption class="wp-element-caption">You&rsquo;ll be able to manage RGB accessories from within Windows with Dynamic Lighting.</figcaption></figure><p class="imageCredit">Microsoft</p></div>



<p>While Microsoft maintains an <a href="https://go.redirectingat.com/?id=111346X1569483&amp;url=https://learn.microsoft.com/windows-hardware/design/component-guidelines/components&amp;xcust=2-1-2004412-1-0-0&amp;sref=https://www.pcworld.com/feed" rel="nofollow">up-to-date list of hardware compatible with Dynamic Lighting</a>, at press time it was still dominated by Razer hardware.</p>



<h2 class="wp-block-heading" id="direct-inking">Direct Inking</h2>



<p>Inking finally works like you&rsquo;d expect it to. On Windows, inking has traditionally meant one of two things: drawing freehand with a pen or scrawling text inside of a dedicated box. If you needed to input text inside of a particular field, you could either ink it (like a signature) or write it inside of a separate input box, where it would be translated into text. </p>



<p>In the real world, you can write on a piece of paper such as a field-trip permission slip and our brains interpret that as text. Now Windows does too. If Windows presents a search box, for example, you can simply write in the search box and Windows will translate your inked handwriting into typed text and enter it in the proper field. Only English is supported, though.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-large"><img alt="Windows 11 23H2 direct inking" class="wp-image-2004448" height="782" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Windows-Ink-direct-inkng.png?w=1200" width="1200" /><figcaption class="wp-element-caption">Direct inking could be a sleeper hit for Surface Pro tablet users.</figcaption></figure><p class="imageCredit">Microsoft</p></div>



<h2 class="wp-block-heading" id="windows-spotlight-tweaks">Windows Spotlight tweaks</h2>



<p><a href="https://www.pcworld.com/article/705519/windows-spotlight-adds-some-fun-back-to-windows-11.html">Windows Spotlight </a>has been one of my favorite features within Windows for years: a simple, gorgeous photo of the natural world which serves as that day&rsquo;s background. Microsoft&rsquo;s tweaking Spotlight further to offer more information about what you&rsquo;re seeing, so you can learn more about it or even potentially plan a visit. </p>



<p>To enable Windows Spotlight on your PC today, right-click on your desktop and choose &ldquo;Personalize&rdquo; and then choose the new Spotlight theme, Microsoft says. Microsoft has never said whether all of its backgrounds are formatted for 4K displays, though I&rsquo;m seeing more and more high-resolution photography on my screen.</p>



<h2 class="wp-block-heading" id="updated-volume-mixer">Updated Volume Mixer</h2>



<p>Microsoft has added an updated volume mixer (<strong>Win+CTRL+V</strong>) where you can configure the volume and source of the audio on a per-app basis, which it <a href="https://www.pcworld.com/article/1531212/windows-11s-new-volume-panel-is-super-helpful.html">showed off earlier</a>. Microsoft is also trying to make it easier for users to find its Windows Sonic spatial sound in addition to other audio enhancement technologies (Dolby DTS and so on) that may be available.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-full"><img alt="Windows 11 23H2 volume mixer" class="wp-image-2004450" height="1167" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Volume-Mixer.png" width="809" /><figcaption class="wp-element-caption">Windows 11&rsquo;s new Volume Mixer.</figcaption></figure><p class="imageCredit">Microsoft</p></div>



<h2 class="wp-block-heading" id="local-file-sharing-updates">Local file sharing updates</h2>



<p>File sharing within Windows has always been a bit of a pain. Now, if you right-click on a file to share it, you&rsquo;ll receive new options. You&rsquo;ll see a button to turn on nearby sharing, for example. </p>



<p>Everyone will now be able to email their files through Outlook directly within the Windows share window and you&rsquo;ll see a search box to type in contact names. Windows will now return suggestions as you do.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-full"><img alt="Windows 11 23H2 local file sharing" class="wp-image-2004447" height="1134" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Windows-Sharing.jpg?quality=50&amp;strip=all" width="894" /><figcaption class="wp-element-caption">File sharing among friends will hopefully get easier under Windows 11.</figcaption></figure><p class="imageCredit">Microsoft</p></div>

<ul>
<li>If you accidentally closed your camera shutter and are beginning a meeting, you&rsquo;ll see a warning that your shutter is closed.</li>



<li>Windows now has new, more natural Narrator voices in Chinese, Spanish (Spain and Mexico), Japanese and English (United Kingdom).</li>



<li>Urgent notifications that are set to break through &ldquo;Do Not Disturb&rdquo; will hide their contents for privacy&rsquo;s sake.</li>

<li>When navigating between desktops in Task View (<strong>WIN + CTRL + left or right arrows</strong>), labels will be shown to show you where you&rsquo;re headed.</li>



<li>Microsoft is adding a new Settings page (<em>Settings &gt; System &gt; Power &amp; battery &gt; Energy recommendations</em>) to help save power by lowering the refresh rate of your display.</li>

## Google, Meta, Microsoft, OpenAI and more agree to voluntary AI safeguards
 - [https://www.pcworld.com/article/2004396/ai-titans-agree-on-safeguards-to-show-that-ai-is-safe.html](https://www.pcworld.com/article/2004396/ai-titans-agree-on-safeguards-to-show-that-ai-is-safe.html)
 - RSS feed: https://www.pcworld.com/index.rss
 - date published: 2023-07-21T17:27:41+00:00

<div id="link_wrapped_content">
<section class="wp-block-bigbite-multi-title"><div class="container"></div></section>



<p>Several of the top American companies developing AI have agreed to work with the U.S. government and commit to several principles to ensure public trust in AI, the White House said Friday.</p>



<p>Amazon, Anthropic, Google, Inflection, Meta, Microsoft, and OpenAI all signed off on the <a href="https://go.redirectingat.com/?id=111346X1569483&amp;url=https://www.whitehouse.gov/wp-content/uploads/2023/07/Ensuring-Safe-Secure-and-Trustworthy-AI.pdf&amp;xcust=2-3-2004396-1-0-0&amp;sref=https://www.pcworld.com/feed" rel="nofollow">commitments</a> to make AI safe, secure, and trustworthy. In May, the Biden administration had said that it would <a href="https://www.pcworld.com/article/1809479/biden-administration-wants-to-use-ai-ethically.html">meet with leading AI developers</a> to ensure that they were consistent with U.S. policy.</p>



<p>The commitments are not binding, and there are no penalties for failing to adhere to them. The policies can&rsquo;t retroactively affect AI systems that have already been deployed, either &mdash; one of the provisions says that the companies will commit to testing the AI for security vulnerabilities, both internally and externally, before releasing it.</p>



<p>Still, the new commitments are designed to reassure the public (and, to some extent, lawmakers) that AI can be deployed responsibly. The Biden administration had already proposed using AI within government to streamline tasks.</p>



<p>Perhaps the most immediate effects will be felt on <a href="https://www.pcworld.com/article/788422/how-to-get-started-with-ai-art-dall-e-mini-ai-dungeon-and-more.html">AI art</a>, as all of the parties agreed to digital watermarking to identify a piece of art as AI-generated. Some services, such as <a href="https://www.pcworld.com/article/1666493/hands-on-with-bing-image-creator-bing-chats-ai-art-tool.html">Bing&rsquo;s Image Creator</a>, already do this. All of the signees also committed to using AI for the public good, such as cancer research, as well as identifying areas of appropriate and inappropriate use. This wasn&rsquo;t defined, but could include the existing safeguards that prevent ChatGPT, for example, from helping to plan a terrorist attack. The AI companies also pledged to preserve data privacy, a priority Microsoft has upheld with <a href="https://www.pcworld.com/article/1997371/microsofts-price-for-its-business-ai-is-insane.html">enterprise versions of Bing Chat and Microsoft 365 Copilot</a>. </p>



<p>All of the companies have committed to internal and external security testing of their AI systems before their release, and sharing information with industry, governments, the public, and academia on managing AI risks. They also pledged to allow third-party researchers access to discover and report vulnerabilities. </p>



<p>Microsoft president Brad Smith <a href="https://go.redirectingat.com/?id=111346X1569483&amp;url=https://blogs.microsoft.com/on-the-issues/2023/07/21/commitment-safe-secure-ai/&amp;xcust=2-3-2004396-1-0-0&amp;sref=https://www.pcworld.com/feed" rel="nofollow">endorsed </a>the new commitments, noting that Microsoft has been an advocate for establishing a national registry of high-risk AI systems. (A <a href="https://www.pcworld.com/article/1481564/congressman-calls-for-a-federal-department-of-ai-to-prevent-skynet.html">California congressman has called for a federal office overseeing AI</a>.) <a href="https://go.redirectingat.com/?id=111346X1569483&amp;url=https://blog.google/technology/safety-security/googles-ai-red-team-the-ethical-hackers-making-ai-safer/&amp;xcust=2-3-2004396-1-0-0&amp;sref=https://www.pcworld.com/feed" rel="nofollow">Google also disclosed its own &ldquo;red team&rdquo; of hackers</a> who try to break AI using attacks like prompt attacks, poisoning data, and more.</p>



<p>&ldquo;As part of our mission to build safe and beneficial AGI, we will continue to&nbsp;<a href="https://go.redirectingat.com/?id=111346X1569483&amp;url=https://openai.com/blog/insights-from-global-conversations&amp;xcust=2-3-2004396-1-0-0&amp;sref=https://www.pcworld.com/feed" rel="nofollow" target="_blank">pilot and refine</a>&nbsp;concrete governance practices specifically tailored to highly capable foundation models like the ones that we produce,&rdquo; OpenAI said in a statement. &ldquo;We will also continue to invest in research in areas that can help inform regulation, such as techniques for assessing potentially dangerous capabilities in AI models.&rdquo;</p>

<p>Alienware&rsquo;s AW3423DWF is an outstanding PC gaming monitor.&nbsp;</p>

<p>This monitor has a QD-OLED panel which is like that used in top-tier OLED televisions. OLED provides a nearly infinite contrast ratio, excellent color performance, and respectable brightness. The result is a punchy, vibrant image with an unparalleled sense of dimensionality. The monitor&rsquo;s ultrawide aspect ratio only heightens the sense of immersion.</p>

<p>Motion clarity is superb. The monitor has a refresh up to 165Hz and supports both AMD FreeSync Premium Pro and VESA Adaptive Sync. The OLED panel technology it uses has lower pixel response times than competitors, too, so there&rsquo;s minimal added blur and ghosting in motion. Motion is as crisp as you&rsquo;ll find below 240Hz.&nbsp;</p>

<p>The Alienware AW3423DWF is an alternative to the AW3423DW, a slightly older and nearly identical monitor. The AW3423DW has a higher fresh rate of 175Hz and supports Nvidia G-Sync Ultimate. It&rsquo;s also $100 to $200 more expensive. Though we recommend the AW3423DWF for most people, the AW3423DW is better for owners of high-end Nvidia graphics cards.</p>

<p>The AW3423DWF is expensive at $1,099.99, but its performance justifies the price. Competing 34-inch and 38-inch ultrawides without QD-OLED technology are often just as expensive but have less impressive image quality.</p>

<h2 class="wp-block-heading" id="what-to-look-for-in-a-gaming-monitor">What to look for in a gaming monitor</h2>



<p>Gamers have unique needs that exceed an average user. Here&rsquo;s what PC gamers should look for in a gaming monitor.</p>



<h3 class="wp-block-heading" id="resolution">Resolution </h3>



<p>Most widescreen gaming monitors have a resolution of 1920&times;1080 (1080p), 2560&times;1440 (1440p), or 4K (3840&times;2160). A higher resolution improves sharpness and clarity, which helps games look more detailed and lifelike. Increasing resolution also increases demand on your video card, however. Gamers with less powerful hardware may want to avoid 4K.&nbsp;</p>



<h3 class="wp-block-heading" id="refresh-rate">Refresh rate</h3>



<p>A higher refresh leads to smoother motion by increasing the number of frames that can appear each second. It also reduces input lag, as each frame appears more quickly. A 144Hz refresh rate is a big improvement over the standard 60Hz, and 240Hz is better still. The improvement becomes more difficult to notice after 240Hz, but 360Hz monitors exist for those who want the lowest input lag possible.</p>



<h3 class="wp-block-heading">DIsplayPort and HDMI 2.<strong>1</strong> </h3>



<p>DisplayPort is the best connection for PC gaming. Even DisplayPort 1.4, which is rather old, can support 4K at 144Hz. HDMI 2.1 can handle 4K at 120Hz and is an acceptable connection for PC gaming, though most gamers will use it for a PlayStation 5 or Xbox Series X|S game console.</p>



<h2 class="wp-block-heading" id="how-we-test-gaming-monitors">How we test gaming monitors</h2>



<p>We test all monitors with a DataColor SpyderXElite calibration tool. This tool can report objective measurements for brightness, contrast, color gamut, color accuracy, color temperature, gamma, and other metrics.&nbsp;</p>



<p>Our results are recorded and compared to the results for past monitors. Though we rely on our eyes for initial impressions, comparing objective results lets us evaluate monitors against hundreds of older models from past reviews and testing.&nbsp;</p>



<p>We examine motion clarity by quickly moving the camera across the map in <em>Civilization VI</em>, playing a round of <em>Rocket League</em>, and panning the camera across the landscape in <em>Final Fantasy XIV</em>&mdash;among other games. Finally, we use the popular <a href="https://go.redirectingat.com/?id=111346X1569483&amp;url=https://www.testufo.com&amp;xcust=2-1-813301-1-0-0&amp;sref=https://www.pcworld.com/feed" rel="nofollow">UFO Test </a>for an apples-to-apples comparison between displays.</p>



<p>In addition to gaming prowess, we consider a monitor&rsquo;s on-screen menu, ergonomic stand (or lack thereof), and overall build quality&mdash;all important qualities irrespective of use case.</p>

<h3 class="wp-block-heading" id="what-resolution-is-best-for-a-gaming-monitor">What resolution is best for a gaming monitor?</h3>



<div class="wp-block-idg-base-theme-faq-answer-block how-to-tip">
<p>Many competitive gamers prefer 1080p resolution because it allows for high frame rates and low input lag even on modest hardware. This also makes it ideal for budget shoppers. 1080p is not as crisp as higher resolutions but, if it allows for higher frame rates, can look good in motion.</p>



<p>1440p remains a great mid-range option. It looks much sharper than 1080p but doesn&rsquo;t drastically increase load on your video card. A variety of 1440p monitors now support a refresh rate up to 240Hz.&nbsp;</p>



<p>4K is the last word in sharpness and clarity. Playing a modern game on a 4K monitor takes the experience to a new level. It&rsquo;s very demanding on your video card, however, so you&rsquo;ll need top-tier hardware for a smooth experience.&nbsp;</p>



<p>Ultrawide monitors differ in resolution because they have a wider screen. Most ultrawide monitors have a resolution of 3440&times;1440, which delivers sharpness similar to a 1440p ultrawide. Some larger monitors have more exotic resolutions: the 49-inch Samsung Odyssey G9 Neo, for example, has 5120&times;1440 resolution.</p>

<div class="wp-block-idg-base-theme-faq-inner-block faq-save-block"><div class="faq-save-content"><span class="faq-rank">2.</span>
<h3 class="wp-block-heading" id="should-i-use-amd-freesync-or-nvidia-g-sync">Should I use AMD FreeSync or Nvidia G-Sync?</h3>



<div class="wp-block-idg-base-theme-faq-answer-block how-to-tip">
<p>AMD FreeSync and Nvidia G-Sync can synchronize the refresh rate of a compatible monitor with the output of an AMD or Nvidia video card. This ensures smooth motion and eliminates screen tearing, a distracting visual artifact.</p>



<p>You might be surprised to learn that most AMD FreeSync and Nvidia G-Sync monitors rely on the VESA Adaptive Sync standard. This is why many gaming monitors now support both. The difference in performance is minimal.</p>



<p>Our <a href="https://www.pcworld.com/article/423202/g-sync-vs-freesync-amd-nvidia-monitor.html">G-Sync vs. FreeSync comparison goes in-depth</a> on their similarities and differences.</p>

<h3 class="wp-block-heading" id="should-i-buy-an-ultrawide-gaming-monitor">Should I buy an ultrawide gaming monitor?</h3>



<div class="wp-block-idg-base-theme-faq-answer-block how-to-tip">
<p><a href="https://www.pcworld.com/article/394848/are-ultrawide-monitors-worth-it.html">The short answer? Yes</a>.&nbsp;</p>



<p>Our top choice, the Alienware AW3423DW, is an ultrawide gaming monitor. An ultrawide aspect ratio is more immersive in racing, simulation, and role-playing games. It also offers a large, more impressive perspective in many strategy games.&nbsp;</p>



<p>Not all games support an ultrawide aspect ratio, however. It&rsquo;s wise to check that your favorite games support ultrawide monitors before making a purchase. Consoles rarely support ultrawide aspect ratios, so console gamers should stick with a widescreen display.</p>

<h2 class="wp-block-heading toc" id="h-die-tucke-der-neuen-technologien">The pitfall of new technologies</h2>



<p>In the past few years, Nvidia has introduced one new technology after another and ultimately (at least currently) won the duel against AMD. With ray tracing, Nvidia has set a graphics trend that will probably be with us for some time to come and in which Team Green is a pioneer. The DLSS upscaling technology is still ahead of the AMD and Intel alternatives.</p>



<p>However, it is becoming more and more of a trend that these innovations are not really backwards compatible&mdash;and that is a disaster from the consumer&rsquo;s point of view. For example, said Deep Learning Super Sampling (DLSS), whose latest version 3 including frame generation is only compatible with Nvidia&rsquo;s current 40-series cards.</p>



<p>So if you bought a 20- or 30-series card in the past years, possibly for a lot of money given the graphics card prices of the last few years, you are now already looking down the tube.</p>



<p>There are of course technical reasons why these new technologies do not work on old cards. <p>It becomes even clearer when comparing FSR against DLSS. While AMD&rsquo;s technology is compatible even with competitor graphics cards from Nvidia and Intel, Nvidia not infrequently excludes GPUs from its own house when it comes to new functions. For example, DLSS only works when using an RTX GPU, older GTX models are left out in the cold, and DLSS 3 Frame Generation only works with the current RTX 40-series graphics cards. AMD relies on industry standards and open source solutions, while Nvidia implements proprietary solutions.</p>



<p>It felt like Nvidia was constantly bringing new features to the market without really addressing the needs of existing customers. A shining example is the built-in video memory, which is becoming increasingly important at higher resolutions and levels of detail.</p>



<p>Only recently, Nvidia has once again shown with the RTX 4060 that the company does not succeed in the balancing act between the use of new technologies and a solid basic configuration. A strong ray tracing performance and DLSS 3 are of little use if the 8GB video memory is no longer sufficient for modern games in full HD. Especially since both technologies further increase the memory requirements.</p>



<p>AMD is not immune to these errors either, as the <a href="https://www.pcworld.com/article/1928214/amd-radeon-rx-7600-review.html">Radeon RX 7600 with only 8 GB of RAM</a> also shows, but it often has a better memory configuration, especially in the midrange and upper class. An illustrative example of this is the <a href="https://www.pcworld.com/article/1431755/amd-radeon-rx-7900-xtx-radeon-rx-7900-xt-review-rdna-3.html">RX 7900 XTX</a>, which not only has significantly more video memory, but is also cheaper than its Nvidia counterpart, the <a href="https://www.pcworld.com/article/1379747/nvidia-geforce-rtx-4080-tested-5-things.html">RTX 4080</a>. And even in terms of pure rasterization performance, AMD cards are often on a par with or even ahead of Nvidia at the same price.</p>

And in most cases, the AMD card will be the cheaper alternative.</p>



<p>Of course, if you value ray tracing and prefer DLSS to AMD&rsquo;s alternative FSR, this question does not arise. Because in these areas Nvidia simply cannot be beaten at the moment, and it is precisely for this reason that the company can also be very creative in its pricing.</p>



<p>AMD, on the other hand, must and wants to convince with classic arguments, namely solid performance, accessible alternative technologies, and ample memory at an attractive price (at least compared to the competition). <p>If you&rsquo;re hunting for a gaming laptop it&rsquo;s tempting to assume your portable of choice will be far too expensive but that&rsquo;s not the case with the Asus TUF Gaming A15 for 2023.</p>



<p>This year&rsquo;s version starts at only $1,199/&pound;1,049 and rises to a reasonable $1,399/&pound;1,499 so you won&rsquo;t have to spend big money.</p>



<p>The model I&rsquo;ve reviewed costs $1,399 in the US but a more palatable &pound;1,199 in the UK. That gets you an Nvidia GeForce RTX 4060 GPU and a 144Hz, 1080p display &ndash; the exact hardware you&rsquo;ll need to run almost every game smoothly for the next few years.</p>



<p>On paper, that&rsquo;s great. But there&rsquo;s more to success than components. Let&rsquo;s dig in.</p>

<p><strong>Further reading: </strong><a href="https://www.pcworld.com/article/436674/the-best-pc-laptops-of-the-year.html">The best PC laptops for any budget</a></p>

<h2 class="wp-block-heading toc" id="design-build">Design &amp; Build</h2>



<ul>
<li>Familiar military design and MIL-STD-810H testing</li>



<li>A solid port selection, albeit without some future-proofed options</li>



<li>Slimmer and lighter than its key rival</li>
</ul>



<p>The Asus TUF range takes inspiration from military hardware, and this 2023 edition still looks like a supply crate. Make of that what you will. </p>



<p>Get beyond that &ndash; and the promotional stickers &ndash; and you&rsquo;ll find a notebook with MIL-STD-810H testing to ensure it can withstand shocks, drops, temperature changes and water spillages.</p>



<p>That said, there&rsquo;s significant movement in the screen, base and keyboard, so I&rsquo;d still recommend a protective sleeve if you frequently deploy the A15 out of the house.</p>



<p>More impressively, the A15 weighs 2.2kg and is 25mm thick, so it&rsquo;s slimmer and lighter than the Acer Nitro 5. This is not a slim and light laptop in the grand, but it&rsquo;s significantly easier to manage.</p>



<figure class="wp-block-pullquote"><blockquote><p>You&rsquo;ll find a notebook with MIL-STD-810H testing to ensure it can withstand shocks, drops, temperature changes and water spillages</p></blockquote></figure>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-large"><img alt="Asus TUF Gaming A15 (2023) " class="wp-image-1989445" height="675" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Asus-TUF-Gaming-A15-09.jpg?quality=50&amp;strip=all&amp;w=1200" width="1200" /></figure><p class="imageCredit">Mike Jennings / Foundry</p></div>



<p>There are full-size USB 3.2 Gen 1 ports on either side, but the USB 4 Type-C port, USB 3.2 Gen 2 Type-C connector and the HDMI, audio and Ethernet outputs are all clustered on the left edge, which may not be handy for your cable-routing or for left-handed users.</p>



<p>The moves to USB 4 and HDMI 2.1 are the only upgrades from the 2022 A15, and the AMD chipset means no Thunderbolt. You&rsquo;ll find that, and one extra USB port, on the Acer.</p>



<p>Elsewhere, the A15 has Wi-Fi 6 rather than 6E, a blurry 720p webcam with no Windows Hello sign-in and no fingerprint reader. There&rsquo;s Gigabit Ethernet but not faster 2.5Gbps. That&rsquo;s expected for an affordable gaming notebook and it won&rsquo;t hinder daily use &ndash; and the Acer suffers similarly.</p>



<h2 class="wp-block-heading toc" id="keyboard-trackpad">Keyboard &amp; Trackpad</h2>



<ul>
<li>A good layout with extra buttons and RGB LED lighting</li>



<li>A soft, squishy typing action that won&rsquo;t satisfy many gamers</li>



<li>An inconsistent trackpad means you&rsquo;ll be better off with a mouse</li>
</ul>



<p>The keyboard could use a little more refinement, too. The buttons may have 1.7mm of travel, but they&rsquo;re soft, without the snappy feel you&rsquo;ll find on the best gaming notebooks.</p>



<p>It&rsquo;s not a disaster; you can use the A15 for gaming and typing. But if you want a robust keyboard to hammer through frantic sessions, you won&rsquo;t get it here.</p>

<p>The trackpad is also soft and its buttons are inconsistent, with differing levels of force required across the pad.</p>



<p>Ergonomically the A15 is ok but not brilliant. The layout is better, with a numberpad, extra audio buttons and single-zone RGB lighting. There&rsquo;s hardly any difference between the Asus A15, the Asus A16, and the Acer &ndash; they&rsquo;re all fine but not amazing.</p>



<h2 class="wp-block-heading toc" id="screen-speakers">Screen &amp; Speakers</h2>



<ul>
<li>A 1080p IPS panel at 144Hz means you&rsquo;ve got the pixels and pace for mainstream games</li>



<li>A lack of brightness means outdoor play is a no-go</li>



<li>Mediocre speakers</li>
</ul>



<p>There are no big additions to the display on this year&rsquo;s A15 &ndash; the 1080p IPS panel with 144Hz refresh rate reappears. You also get G-Sync support alongside Nvidia Advanced Optimus and a MUX switch so it&rsquo;s easy to use the laptop on mains and battery power.</p>



<p>That&rsquo;s fine for mainstream gaming and esports, but this screen isn&rsquo;t fast enough for high-end competition. It doesn&rsquo;t have the extra immersion you&rsquo;ll get from 16:10 panels or screens with higher resolutions.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-large"><img alt="Asus TUF Gaming A15 (2023) " class="wp-image-1989448" height="675" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Asus-TUF-Gaming-A15-02.jpg?quality=50&amp;strip=all&amp;w=1200" width="1200" /></figure><p class="imageCredit">Mike Jennings / Foundry</p></div>



<p>In terms of quality, there&rsquo;s good and bad news. Positively, the A15&rsquo;s delta E of 1.46 means colors are accurate and the contrast ratio of 935:1 is fine for gaming &ndash; mainstream games will look good here.</p>



<p>Negatively, though, the maximum brightness of just 290 nits is not high enough for outdoor use. And it&rsquo;s always quite annoying when other specifications aren&rsquo;t available &ndash; the 2022 model could sometimes be found with 1440p panels and 300Hz refresh rates but there&rsquo;s no sign of that this year.</p>



<p>Unsurprisingly, you get mediocre speakers on this unit, with non-existent bass and a tinny top-end. Use a headset instead and you&rsquo;ll have a far better time.</p>



<h2 class="wp-block-heading toc" id="specs-performance">Specs &amp; Performance</h2>



<ul>
<li>AMD Ryzen 7 or Ryzen 9 processors</li>



<li>Nvidia GeForce RTX 4050, 4060 or 4070 graphics</li>



<li>16GB DDR5 memory and 512GB SSD</li>
</ul>



<p>The Nvidia GeForce RTX 4060 GPU has 8GB of memory and a TDP of 140W, which is right at the top of the GPU&rsquo;s thermal envelope. Asus has deployed an AMD Ryzen 7 7735HS processor with eight multi-threaded cores and a boost speed of 4.75GHz.</p>



<p>You&rsquo;ll find that processor in the TUF A16, too, but it&rsquo;s odd when most other gaming laptops &ndash; including the Acer Nitro 5 &ndash; include Intel chips.</p>



<figure class="wp-block-pullquote"><blockquote><p>You have enough pace to handle any top single-player game for the next few years</p></blockquote></figure>



<p>The A15 has 16GB of dual-channel memory and a 512GB SSD. That memory is fine, but the SSD is small considering the size of modern games. Thankfully there&rsquo;s a second SSD slot inside.</p>



<p>The Asus played CyberPunk 2077 at Ultra settings with an 88fps average, and returned a still-playable result of 55fps with Ray Tracing and DLSS. A 1080p average of 85fps in Far Cry: New Dawn and a mighty 247fps result in Rainbow Six Siege continued the A15&rsquo;s good form.</p>



<p>You have enough pace to handle any top single-player game for the next few years, especially if you don&rsquo;t want to run the top settings. There&rsquo;s easily enough speed for any eSports game too.</p>


<div class="extendedBlock-wrapper block-coreImage undefined"><figure class="wp-block-image size-large"><img alt="Asus TUF Gaming A15 (2023) " class="wp-image-1989449" height="675" src="https://b2c-contenthub.com/wp-content/uploads/2023/07/Asus-TUF-Gaming-A15-07.jpg?quality=50&amp;strip=all&amp;w=1200" width="1200" /></figure><p class="imageCredit">Mike Jennings / Foundry</p></div>



<p>In 3D Mark Time Spy, the A15 scored 10,020 points &ndash; nearly 1,000 beyond the RX 7600S inside the A16. That result rose by 600 points in Turbo mode. That&rsquo;s as good as it gets for the laptop version of the RTX 4060.</p>



<p>AMD&rsquo;s processor does a decent job against Intel&rsquo;s chips. In PCMark 10 the AMD CPU scored 7,132, which outpaces the 6,926 scored by the i5-12500H in the Acer and the i7-12700H, which has actually proved a little slower in PCMark 10. The i5-13500H and i7-13700H don&rsquo;t make a huge difference here, either.</p>



<p>In Cinebench R20&rsquo;s single- and multi-threaded tests, the A15 scored 579 and 5,241, with the former behind Intel and the latter ahead.</p>



<p>The Ryzen chip&rsquo;s Geekbench 5 score of 9,291 is about 3,000 points shy of the Core i7-12700H and not far beyond the i5-12500H. Its Geekbench 6 result of 10,120 sits behind the i7, which is more than 1,000 points better in that test.</p>








<figure class="wp-block-pullquote"><blockquote><p>Remarkably, the A15 lasted for almost nine hours when working with the display at full brightness</p></blockquote></figure>



<p>For everyday browsing, photo-editing, multi-tasking and office tasks the AMD chip is great, but if you want true content creation ability you&rsquo;d be better off with Intel i7-12700H.</p>



<p>More positively the Asus never proved too hot or loud. Its fans only produced a tiny whirr in everyday Performance Mode and the noise wasn&rsquo;t much more intrusive in Turbo Mode.</p>



<p>You even get decent battery life. I got two hours of gaming from the TUF A15 &ndash; twice as much as the Nitro 5. Remarkably, the A15 lasted for almost nine hours when working with the display at full brightness and twelve hours when running a film with the brightness reduced to our normal test setting.</p>



<p>The A16 exhibited similarly lengthy results, so there&rsquo;s clearly something good happening in the 90Wh power pack and this is a laptop that&rsquo;s certainly usable for a full working day.</p>

<h3 class="wp-block-heading" id="pcs-vs-chromebooks-which-is-better-for-school">PCs vs. Chromebooks: Which is better for school?</h3>



<div class="wp-block-idg-base-theme-faq-answer-block how-to-tip">
<p>We have a lengthy overview of the advantages of <a href="https://www.pcworld.com/article/402141/chromebooks-versus-windows-laptops-which-should-you-buy.html">Chromebooks versus laptops</a>, but the simple answer to this question is: whatever your school district tells you. </p>



<p>A Chromebook is better for school if your child will be working &ldquo;online,&rdquo; meaning in the classroom but on a website that&rsquo;s authorized by the teacher. A PC is more useful if your child will be using apps, such as Minecraft for Education, or transitioning from elementary school to middle school, or from middle school to high school. The extra flexibility to run apps (and, yes, games) will be appreciated by your child. Colleges and universities almost always ask their students to bring PCs or Macs with them, so a laptop is most appropriate.</p>

<div class="wp-block-idg-base-theme-faq-answer-block how-to-tip">
<p>Dorm rooms typically prioritize the efficient use of space, so laptops will be the most typical. Any way that you can use the extra space, though, is useful. A high-resolution display may mean that you can more effectively &ldquo;fit&rdquo; more information into the space with smaller fonts. But a monitor <em>mount</em>, like we&rsquo;ve included in our list of monitors, can clamp on to a desk and allow for a second display, which vastly increases productivity and can also be used to stream video for friends, too. Our back-to-school list of monitors includes both cheap 1080p options as well as 4K displays and even gaming displays for after hours.</p>
